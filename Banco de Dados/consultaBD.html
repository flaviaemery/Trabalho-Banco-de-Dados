Sobre

A ideia incial do trabalho, foi a criação de uma biblioteca virtual, onde, os livros seriam separados por gênero literário e leitura técnica, porém, a dificuldade em encontrar arquivos csv disponíveis para a execução do trabalho, fez com que houvesse uma mudança para arquivos de música, definindo artista/banda com álbum, ano de lançamento do álbum, gênero musical e subgênero musical.

A coleta de dados para exibição de um relatório, foi feita através da conexão do bnco de dados em um arquivo php.

A página em HTML foi retirada de um modelo do W3schools.com e sofreu as modificações para o trabalho.

A página Referência foi um arquivo csv do GitHubGist do Jaidevd/boos.csv. O arquivo foi importado para o Banco de dados através do phpmyadmin.

  • O primeiro FORM apresentado para a busca no banco de dados (Apresentação do BD musica), mostra dados através de:

    • SELECT label_al, title, year, to_ed, label_ed FROM Albums A INNER JOIN Edges E WHERE A.id_al=E.from_ed;.

  • O segundo FORM apresentado para a busca no banco de dados (Album e Gênero Rock), mostra dados através de:

    • SELECT title, name_gen FROM Albums Al INNER JOIN Edges E ON (Al.id_al = E.from_ed) INNER JOIN Genre Ge ON (Ge.id_gen = E.to_ed) WHERE name_gen LIKE '%Rock%';.

  • O terceiro FORM apresentado para a busca no banco de dados (Artista(ordenação)/Album(Alias)/Ano(Alias)), mostra dados através de:

    • SELECT name_ar AS Artista, title AS Album, year AS Ano FROM Artist Ar INNER JOIN Edges E ON (Ar.id_ar = E.to_ed) INNER JOIN Albums Al ON (Al.id_al = E.from_ed) ORDER BY name_ar;.

  • O quarto FORM apresentado para a busca no banco de dados (Album/Gênero/Subgênero(UNION)), mostra dados através de:

    • SELECT title, name_gen FROM Albums Al INNER JOIN Edges E ON (Al.id_al = E.from_ed) INNER JOIN Genre Ge ON (Ge.id_gen = E.to_ed) UNION SELECT title, name_sub FROM Albums Al INNER JOIN Edges E ON (Al.id_al = E.from_ed) INNER JOIN Subgenre Sub ON (Sub.id_sub = E.to_ed);.

  • O quinto FORM apresentado para a busca no banco de dados (Artista/Album entre 1990 e 2001), mostra dados através de:

    • SELECT name_ar, title, year FROM Artist Ar INNER JOIN Edges E ON (Ar.id_ar = E.to_ed) INNER JOIN Albums Al ON (Al.id_al = E.from_ed) WHERE year >= '1990' AND year <= '2001' ORDER BY year desc;.

  • O sexto FORM apresentado para a busca no banco de dados (Album/Subgênero), mostra dados através de:

    • SELECT title, name_sub FROM Albums Al INNER JOIN Edges E ON (Al.id_al = E.from_ed) INNER JOIN Subgenre Sub ON (Sub.id_sub = E.to_ed);.

  • O sétimo FORM apresentado para a busca no banco de dados (Album e Gênero musical), mostra dados através de:

    • SELECT title, name_gen FROM Albums Al INNER JOIN Edges E ON (Al.id_al = E.from_ed) INNER JOIN Genre Ge ON (Ge.id_gen = E.to_ed);.

  • O oitavo FORM apresentado para a busca no banco de dados (Artista-Album-Ano < 1960), mostra dados através de:

    • SELECT name_ar AS Artista, title AS Album, year AS Ano FROM Artist Ar INNER JOIN Edges E ON (Ar.id_ar = E.to_ed) INNER JOIN Albums Al ON (Al.id_al = E.from_ed) HAVING year <= '1960' ORDER BY year desc;.
